# Plantilla LaTeX para tesis

Estructura base para el trabajo de investigación.

## Archivos principales
- `main.tex`: punto de entrada.
- `config.tex`: datos de portada y metadatos.
- `preamble.tex`: paquetes y configuración global.
- `preliminares/`: portada y preliminares.
- `capitulos/`: capítulos del cuerpo principal.
- `cierre/`: bibliografía.
- `anexos/`: anexos.
- `referencias/references.bib`: referencias bibliográficas.
- `recursos/logo/logo.png`: logotipo institucional.

## Compilación
Esta plantilla está preparada para XeLaTeX o LuaLaTeX porque usa `fontspec` y define Arial como fuente principal.
