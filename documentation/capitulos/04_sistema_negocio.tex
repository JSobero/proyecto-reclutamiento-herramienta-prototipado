\chapter{Sistema de Negocio}
\section{Descripción del Sistema de Negocio}
El sistema de negocio del proceso de “Concurso de Personal” en la empresa Tambo comprende el conjunto de actividades, actores y recursos involucrados en la gestión del reclutamiento y selección de personal. Este sistema inicia con la identificación de la necesidad de contratación y la publicación de vacantes, continúa con la recepción y evaluación de postulantes, y culmina con la selección, contratación y entrenamiento inicial del candidato idóneo.
En este contexto, participan diversos actores como el postulante, el reclutador y el área de recursos humanos, quienes interactúan mediante procesos definidos para garantizar la incorporación eficiente de talento humano a la organización.
Para que este proceso funcione, el sistema de negocio se apoya en los siguientes recursos fundamentales:
\begin{itemize}
    \item \textbf{Recursos Tecnológicos (Hardware):} Computadoras de escritorio o laptops ubicadas en la Sede Central (para el equipo de Atracción de Talento) y computadoras de tienda o dispositivos móviles (para los Administradores de Tienda). Acceso a conexión a internet estable.
    \item \textbf{Recursos Lógicos (Software):} Plataforma web de gestión de talento, repositorio de base de datos centralizada, y sistemas de correo electrónico corporativo.
    \item \textbf{Recursos Físicos:} Instalaciones de las tiendas Tambo (usadas para las entrevistas presenciales y la etapa de entrenamiento práctico) y la Sede Central administrativa.
\end{itemize}

El sistema actual presenta limitaciones relacionadas con la alta carga operativa, procesos manuales y falta de integración de la información, lo que genera retrasos y posibles pérdidas de talento.

\section{Alcance del Sistema}

El sistema abarcará las siguientes funcionalidades:

\begin{itemize}
    \item Publicación de convocatorias laborales en el portal del sistema.
    \item Registro y almacenamiento de postulaciones con validación de datos.
    \item Pre-calificación automática de candidatos mediante filtros lógicos configurables.
    \item Gestión de estados del candidato a lo largo de todo el pipeline.
    \item Selección y contratación de personal.
    \item Gestión y seguimiento de la fase de inducción (Trainee), registrando el inicio y la culminación del entrenamiento del nuevo colaborador en tienda.
\end{itemize}

El sistema no incluirá:
\begin{itemize}
    \item Gestión de desempeño a largo plazo posterior a la contratación definitiva (evaluaciones anuales, KPIs del colaborador).
    \item Procesos de nómina, cálculo de horas extras, liquidaciones o gestión salarial.
\end{itemize}
