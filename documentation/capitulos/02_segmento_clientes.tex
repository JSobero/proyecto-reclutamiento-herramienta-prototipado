\chapter{Segmento de Clientes}
Para definir con precisión el público objetivo de la solución, se ha elaborado el Perfil del Cliente
basado en las necesidades reales del área de Atracción de Talento de Tambo.

\section{ Customer Jobs (Trabajos del cliente)}
\textbf{Funcionales:} Publicar vacantes masivas para operarios de tienda y filtrar perfiles
que cumplan con los requisitos mínimos de contratación.

\textbf{Sociales:} Mantener la reputación de Tambo como marca empleadora mediante una
comunicación profesional y oportuna con los postulantes.

\textbf{Emocionales:} Experimentar seguridad y tranquilidad al cumplir con las cuotas
semanales de contratación para las aperturas de nuevas tiendas.

\section{Pains (Dolores)}
\begin{itemize}
  \item \textbf{Frustraciones:} Invertir tiempo excesivo revisando currículos no alineados con el perfil buscado.
  \item \textbf{Obstáculos:} Dificultad para coordinar entrevistas presenciales con candidatos ubicados lejos de las sedes laborales.
  \item \textbf{Riesgos:} Pérdida de talento valioso por procesos lentos o falta de seguimiento efectivo.
\end{itemize}

\section{Gains (Ganancias)}
\begin{itemize}
  \item \textbf{Resultados esperados:} Reducir el tiempo total del ciclo de contratación, desde la convocatoria hasta la firma del contrato.
  \item \textbf{Resultados deseados:} Disponer de unabasededatosdigitalizadayaccesibleparafuturasvacantes en zonas específicas.
\end{itemize}
